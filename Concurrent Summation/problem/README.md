# Concurrent Summation

**Problem Statement:**
Write a C program that accepts integers as command-line arguments. The program must create a separate subprocess for each argument to calculate and print the sum of its digits. If no arguments are provided, the program must print "Error: no argument found" to STDOUT and exit with code 0. 

**Example Execution:**
```bash
./a.out 1245 12514 15125 12415 1512

```

**Expected Output:**

```text
12 13 14 13 9

```
